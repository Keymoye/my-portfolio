## ğŸ§© Installation & Setup

1. **Clone the repository**
```bash
git clone https://github.com/Keymoye/my-portfolio.git
cd my-portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Run locally**
